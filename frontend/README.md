# JW Progress Tracker - Frontend

A Progressive Web App (PWA) for tracking daily spiritual activities.

## Features

✅ **Daily Text Tracker** - Read and track daily text from "Examining the Scriptures Daily"
✅ **Bible Reading Tracker** - Follow "Read the Bible in a Year" schedule
✅ **Weekly Meeting Tracker** - Prepare for midweek and weekend meetings
✅ **Progress Statistics** - View streaks and completion rates
✅ **Offline Support** - Works without internet connection
✅ **Mobile-First Design** - Optimized for smartphones

## Quick Start

\`\`\`bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
\`\`\`

## Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS + DaisyUI
- **State Management:** Zustand with persistence
- **Routing:** React Router v6
- **PWA:** vite-plugin-pwa with Workbox
- **Icons:** Lucide React
- **Date Handling:** date-fns
